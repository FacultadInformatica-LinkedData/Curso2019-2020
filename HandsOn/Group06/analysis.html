ANÁLISIS DE LOS CSV

1. CSV MUSEOS

NOMBRE TIPO RANGO DESCRIPCIÓN
PK Integer [0-9] Identificación
NOMBRE String [a-z][A-Z] Nombre del museo
DESCRIPCIÓN-ENTIDAD String [a-z][A-Z][0-9] Pequeña descripción del museo que puede contener cualquier dato
HORARIO String [a-z][A-Z][0-9] Dias de apertura con su horario de visitas
EQUIPAMIENTO String [a-z][A-Z][0-9] Contenido de los museos como por ejemplo si tienen visitas guiadas
TRANSPORTE String [a-z][A-Z][0-9] Como llegar al museo
DESCRIPCIÓN String [a-z][A-Z][0-9] Datos sobre el museo
ACCESIBILIDAD Integer [0-3] Grado de accesibilidad que tiene el museo
CONTENT-URL String [a-z][A-Z][0-9] URL del museo
NOMBRE-VIA String [a-z][A-Z][0-9] Nombre que tiene una calle, vía o avenida entre otros
CLASE-VIAL String [a-z][A-Z] Tipo de via
TIPO-NUM Integer [A-Z] Tipo de la calle, vía o avenida entre otros
NUM Integer [0-9] Número de la calle, vía o avenida entre otros
PLANTA String [A-Z][0-9] Número de piso del edificio
ESCALERA Integer [a-z][A-Z][0-9] Número de escalera
ORIENTACIÓN String [a-z][A-Z][0-9] Hacia que punto cardinal está orientado (NOrte, Sur, Este u Oeste)
LOCALIDAD String [a-z][A-Z] Localidad del museo
PROVINCIA String [a-z][A-Z] Provincia donde se encuentra el museo
CÓDIGO-POSTAL Integer [0-9] Código postal al que pertenece el museo
BARRIO String [a-z][A-Z] Barrio al que pertenece dentro de su localidad
DISTRITO String [a-z][A-Z] Parte de la localidad del museo
COORDENADA-X Integer [0-9] Coordenada x del museo
COORDENADA-Y Integer [0-9] Coordenada y del museo
LATITUD Coordinate N/A Latitud con respecto a la tierra
LONGITUD Coordinate N/A LOngitud con respecto a la tierra
TELÉFONO Integer [0-9] Número de teléfono del museo
FAX Integer [0-9] Número de fax del museo
EMAIL String [a-z][A-Z] Email que corresponde de cada museo
TIPO String [a-z][A-Z] Dentro de que entidad u organismo pertenece

ANÁLISIS DE LA LICENCIA

Las condiciones generales permiten la reutilización de los documentos para fines comerciales y no comerciales. Se entiende por reutilización el uso de documentos que obran en poder del Ayuntamiento de Madrid, siempre que dicho uso no constituya una actividad administrativa pública.
