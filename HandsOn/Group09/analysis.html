Data Type Description
CODIGO Integer Identificador numerico de las bibliotecas
NOMBRE String Nombre de la biblioteca
INSTITUCION String Institucion a la que pertenece la biblioteca
DIRECCION String Calle y numero donde se encuentra
CODIGO_POSTAL Integer Codigo Postal de la biblioteca
LOCALIDAD String Ciudad o pueblo donde se encuentra
PROVINCIA String Si se encuentra en España, provincia en la que se encuentra
COMUNIDAD_AUTONOMA String Si se encuentra en España, comunidad en la que se encuentra
PAIS String Tipo de numeracion de la via
FECHA_FUNDACION Integer Año de fundacion de la biblioteca
HORARIO String Horario de apertura de la biblioteca
TIPO_ACCESO String Tipo de acceso de la biblioteca, restringido o libre
NOTAS_ACCESO String Si el acceso es restringido, comentarios sobre el tipo de restriccion
TIPO_BIBLIOTECA Integer Si la biblioteca es especializada o general
TIPO_DEPENDENCIA String Quien esta al cargo de la biblioteca
PRESTAMO_INTERBIBLIOTECARIO String No se va a utilizar
CATALOGOS_IMPRESOS String No se va a utilizar
OBSERVACIONES String No se va a utilizar
TRANSPORTE String No se va a utilizar
