Data Analysis:

We have this data for our resources:

Field Type Comments
Expedient Number int This variable can take any number, expedient numbers normally are long.
Date date The date when the accident was produced with format dd/mm/yyyy.
Hour date The hour when the accident was produced, format is 24h.
Address String Saves the postal address where the accident was produced.
Number int Can take any value always its a number, it can take an empty value, its the concrete number inside the street where the accident was produced, its just to locate the accident better.
District String The district where the accident was produced, the address of the accident must be inside the district limits, is a trivial thing.
Type of accident String Its a string that indicates what type the accident was.
Weather String It's a string that indicates the weather at the time of the accident.
Type of person String It's a string that indicates the person who was affected by the accident, it can take two values: driver (of the bicycle) or pedestrian, the case is normally duplicated with the same expedient number, date, time, address etc if it was a hit between a pedestrian and a driver, one being driver as type of person and the other one being pedestrian.
Age range string It's not an integer cause on the dataset is setted as a range between 2 near values.
Genre String It tells the genre of the affected by the accident, it can be male or female.
Injury int It describes the injury grade of the affected person on the accident, it can take empty values.
